Outbreak API

This site is a collection of APIs developed from the Su / Wu Labs at Scripps Research on COVID-19-related data and resources. These APIs provide all the data at outbreak.info(link), which is a resource to collect, share, and integrate COVID-19 and SARS-CoV-2 data critical to for scientific research.

They were built using BioThings SDK and follow the same design pattern of our official BioThings APIs (MyGene.info, MyVariant.info, MyChem.info, etc). These APIs are also pending to be integrated into the proper official BioThings API based on their entity type.
